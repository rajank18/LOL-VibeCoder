## ğŸ“– How It Works

### 1. Repository Analysis
- **Cloning**: Automatically clones GitHub repositories (tries main, master, develop branches)
- **File Detection**: Identifies code files across multiple languages
- **Static Analysis**: Analyzes code structure, comments, and patterns

### 2. AI Pattern Detection
- **Generic Names**: Detects AI-generated variable/function names
- **Perfect Formatting**: Identifies overly consistent code formatting
- **Boilerplate Code**: Finds template-like code patterns
- **Repetitive Patterns**: Detects code repetition typical of AI generation

### 3. Advanced Scoring
- **Node.js Analysis**: Basic metrics (files, lines, comments, structure)
- **Python Analysis**: Advanced static analysis (naming quality, complexity, tests)
- **LLM Integration**: Google Generative AI for intelligent final scoring
- **Combined Metrics**: Merges all analysis results for comprehensive scoring

### 4. Final Verdict
- **Overall Score**: 0-10 scale (higher = more AI-generated)
- **Category Scores**: Individual scores for different aspects
- **Verdict**: Clear determination with confidence level
- **Highlights**: Detailed analysis findings
